# 🚀 VodsInvite - Bot de Gestion d'Invitations Premium

VodsInvite est un bot Discord complet conçu pour automatiser la gestion des invitations, récompenser l'activité vocale et animer votre communauté avec un système de clans (Teams).

## ✨ Fonctionnalités Principales

*   **📈 Tracking d'Invitations Automatique** : Plus de commandes complexes. Créez simplement vos liens via Discord, le bot s'occupe du reste avec un système "Zéro Délai".
*   **💰 Économie Intégrée** : Gagnez des pièces en invitant de nouveaux membres ou en restant actif dans les salons vocaux.
*   **🏆 Paliers de Récompenses** : Débloquez automatiquement des rôles exclusifs (Bronze, Silver, Gold, Platinum) en fonction de vos points.
*   **🛡️ Système de Teams** : Créez votre clan, gagnez de l'XP collective et débloquez des salons vocaux privés et des bonus de gains.
*   **🎯 Quêtes & Défis** : Relevez des défis quotidiens et participez à des compétitions entre teams pour booster vos gains.
*   **⚙️ Panneau d'Administration** : Configuration simplifiée via une interface interactive pour gérer les rôles, la sécurité et les logs.

## 🛠️ Installation

1.  Clonez ce dépôt.
2.  Installez les dépendances : `npm install`
3.  Configurez votre fichier `.env` (voir `.env.example`).
4.  Lancez le bot : `node bot/index.js`

## 📝 Commandes Utiles

*   `/invites stats` : Voir vos performances d'invitation.
*   `/economy menu` : Accéder à votre solde et à la boutique.
*   `/team create` : Lancer votre propre clan.
*   `/admin panel` : (Admin uniquement) Configurer le bot.
